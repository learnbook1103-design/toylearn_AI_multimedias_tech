## 문제상황
```
페르소나: 뉴미디어 스타트업의 '숏폼 콘텐츠 PD'

배경: 당신은 IT 및 테크 트렌드를 소개하는 유튜브 채널을 운영하는 스타트업 '테크인사이드'의 PD입니다. 최근 회사는 틱톡과 릴스 유입을 늘리기 위해 '60초 지식 요약' 시리즈를 런칭했습니다.

핵심 문제: 마케팅 팀으로부터 "생성형 AI의 미래와 윤리적 쟁점"이라는 주제의 10페이지짜리 심층 분석 리포트(PDF)를 전달받았습니다. 내용은 매우 전문적이고 딱딱한 텍스트 위주여서, 이를 그대로 읽으면 시청 지속 시간이 3초 미만으로 떨어질 것이 확실시되는 상황입니다.
```

## 문제 해결 프롬프트
```
역할: 당신은 뉴미디어 스타트업 '테크인사이드'의 숏폼 콘텐츠 PD입니다.
목표: 제공된 '생성형 AI의 미래와 윤리적 쟁점' 보고서 전문의 내용을, 초등학생도 이해할 수 있는 수준으로 변환하여 60초 분량의 숏폼 영상 촬영 대본을 완성해야 합니다.

주제 핵심 내용 (보고서 기반 필수 포함):

1.  기술 진화: LLM의 멀티모달 통합과 실시간 생성 능력 (Intro에 반영).
2.  윤리적 쟁점 (3가지 모두 본론에 반영):
      * 저작권 침해: 학습 데이터 수집의 투명성 부족 및 보상 메커니즘 미비 문제.
      * 할루시네이션: 거짓 정보를 사실처럼 생성하여 정보 신뢰성을 훼손하는 현상.
      * 편향성: 학습 데이터의 편견이 반영되어 차별적 콘텐츠를 양산하는 문제.
3.  해결책: 'Human-in-the-loop (인간 개입)' 시스템을 통한 최종 검수 강화 (결론에 반영).

요구 사항:

1.  후킹 멘트 (초반 3초): 할루시네이션(거짓 정보)을 활용하여 시청자의 이목을 즉각적으로 끌 것.
2.  구성: 훅(Hook), 본론(3가지 쟁점), 결론(Human-in-the-loop CTA)의 3단 구성을 따르십시오.
3.  대본 요소: 화면 구성(Shot/Visual), 자막, 내레이션(PD 멘트), 예상 시간(초), 효과음(SFX)을 구체적으로 분리하여 제시하십시오.

출력 형식: 모든 내용을 다음의 JSON 형식으로만 출력하십시오.
```
