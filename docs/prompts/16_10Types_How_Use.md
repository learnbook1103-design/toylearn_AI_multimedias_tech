## 문제 상황
- 최근 1개월간 회사 유튜브 채널에 업로드한 신규 영상들의 '평균 시청 지속 시간'이 이전 분기 대비 30% 급감했다

## 프롬프트 유형 선택 이유
- 역할부여(Persona) : 문제를 정확히 진단하고 해결하기 위해, '데이터 분석가'와 '유튜브 전문가'의 전문 지식과 관점을 동시에 요구해야 합니다.
- 체인 오브 스루트(CoT) : 최정족인 '해결책'을 도출하기 전에, '문제 정의'와 '원인 분석'이라는 논리적인 추론 과정을 강제하여 답변의 신뢰성과 깊이를 확보해야 합니다.

## 필요 요건
| 구분 | 필요 요건 (핵심 질문) | 이유 (왜 이 정보가 필요한가) |
| :--- | :--- | :--- |
| I. 내부 영상 데이터 | 1. 이전 분기 대비 평균 영상 길이 변화 | 시청 지속 시간 '절대치'가 아닌 '비율'이 급감했더라도, 영상 길이가 길어졌다면 시청자의 부담이 커져 급감이 발생했을 수 있습니다. (예: 5분 영상 -> 10분 영상) |
| | 2. 주요 이탈 구간 (Audience Retention) | 시청자가 가장 많이 이탈하는 '시간대' (예: 도입 10초, PPL 구간, 결론)를 알아야 문제의 원인이 '흥미 유발 실패'인지 '중간 편집 실패'인지 정확히 진단 가능합니다. |
| | 3. 시청 지속 시간 우수 영상의 공통점 | 급감 기간 중에도 시청 지속 시간이 잘 유지된 영상이 있다면, 그 영상의 주제, 포맷, 출연진 등의 공통점을 추출하여 성공적인 포맷으로 복귀하는 근거를 마련해야 합니다. |
| II. 콘텐츠 속성 정보 | 4. 최근 콘텐츠 포맷 및 주제의 변화 | 급감 직전 1개월 동안 영상 스타일(편집 톤, 음악, 자막)이나 주요 주제가 이전 분기와 다르게 파격적으로 변경되었는지 확인해야 합니다. |
| | 5. 최근 시청자 피드백 (댓글, 커뮤니티) | "지루해졌다", "너무 빨라졌다/느려졌다" 등 시청자의 정성적인 불만을 수집하여, 데이터 지표가 포착하지 못하는 콘텐츠의 '매력' 문제를 진단해야 합니다. |
| III. 외부 유입 및 시장 요인 | 6. 유입 경로 및 노출 클릭률(CTR) 변화 | 시청자가 어떤 경로로 유입되고 있는지, 그리고 썸네일과 제목의 매력도를 나타내는 CTR이 이전 대비 어떻게 변했는지 확인해야 콘텐츠 유입 전략의 문제점을 파악할 수 있습니다. |
| | 7. 경쟁사 채널 콘텐츠 및 성과 | 우리가 급감했을 때 경쟁사 채널은 어떤 주제와 포맷으로 시청 지속 시간을 유지/향상했는지 분석하여, 시장의 요구가 어디로 이동했는지 파악해야 합니다. |

## 문제 해결 프롬프트
```
당신은 20년 경력의 유튜브 채널 성장 컨설턴트이자 데이터 분석가입니다.

우리가 해결해야 할 문제 상황은 다음과 같습니다:
- 문제 발생: 최근 1개월간 회사 유튜브 채널에 업로드된 신규 영상들의 '평균 시청 지속 시간'이 이전 분기 대비 30% 급감했습니다.
- 최종 목표: 시청 지속 시간을 이전 수준으로 회복시키고, 채널의 장기적인 성장을 위한 다음 분기 콘텐츠 전략을 수립해야 합니다.

[지시 사항 - 체인 오브 스루트 적용]

데이터가 부족한 상황이므로, 다음의 3단계 분석 절차를 반드시 거쳐 최종 전략을 제시해 주세요.

1단계: 문제의 핵심 원인 진단 (데이터 분석가 관점)
- 시청 지속 시간이 30% 급감했을 때, 유튜브 데이터 분석 관점에서 가장 먼저 확인해야 할 핵심 지표 2가지와 그 지표를 통해 확인할 수 있는 가장 유력한 원인 가설 2가지를 제시하십시오. (예: '도입부 이탈률', '유입 경로의 변화' 등)

2단계: 가설 기반의 콘텐츠/포맷 문제점 예측 (컨설턴트 관점)
- 1단계에서 제시한 원인 가설 2가지 중, 현재 상황에서 가장 해결이 시급한 원인 가설 1가지를 선택하십시오.
- 이 선택된 원인이 '콘텐츠의 포맷, 편집, 또는 주제' 중 어떤 부분에서 가장 크게 나타났을 것인지 예측하고 그 이유를 설명하십시오.

3단계: 다음 분기 실행 전략 도출
- 2단계의 문제점 예측을 해결하기 위한 다음 분기 유튜브 채널 운영 및 마케팅 전략 3가지를 구체적으로 제시하십시오. 각 전략은 반드시 '시청 지속 시간 30% 회복'을 최우선 목표로 삼아야 합니다.
```

## 프롬프트 실행 결과
---
### 1단계: 가상의 사실 (SWOT 요소) 설정

| 구분 | 가상의 사실 |
| :--- | :--- |
| S (강점) | 강력한 '커뮤니티/소셜 기능': 사용자 간 상호작용이 경쟁사 대비 훨씬 활발하며, 사용자 이탈률이 낮음. (높은 리텐션) |
| W (약점) | 저조한 초기 인지도 및 높은 사용자 획득 비용(CAC): 브랜딩이 약해 마케팅 효율이 낮으며, 유료 광고에 쉽게 노출되지 않음. |
| O (기회) | 새로운 '숏폼 콘텐츠' 플랫폼의 등장 및 연동 가능성: 마케팅 비용이 상대적으로 저렴한 신규 플랫폼에서 우리의 소셜 기능과 연계하여 바이럴을 일으킬 기회가 생김. |
| T (위협) | 경쟁사의 대규모 '가격 할인 및 캐시백' 공세: 경쟁사가 다음 분기 동안 대규모 자본을 투입하여 파격적인 가격 혜택으로 신규 사용자를 대거 흡수 중. |

---

### 2단계: SWOT 매트릭스 완성 및 핵심 전략 축 선정

| 구분 | 내부 (S: 커뮤니티 강점 / W: 낮은 인지도/고비용) |
| :--- | :--- |
| 외부 (O: 숏폼 플랫폼 연계 / T: 경쟁사 가격 공세) | SO: 강점으로 기회 활용 | ST: 강점으로 위협 극복 |
| WO: 약점 보완하며 기회 활용 | WT: 약점을 줄이고 위협 회피 |

핵심 전략 축 선택: ST 전략 (강점 - 위협 극복)

선택 이유:
현재 가장 시급한 것은 T (경쟁사 가격 공세)를 정면으로 대응하는 것입니다. W (낮은 인지도/고비용)를 가진 상태에서 WO 전략을 펼치기엔 시간이 부족합니다. 따라서, 우리 앱의 핵심 자산인 S (강력한 커뮤니티)를 활용하여, '가격' 외의 가치 (소속감, 상호작용)를 강조함으로써 경쟁사의 가격 공세(T)를 무력화하는 전략이 사용자 증가세 회복에 가장 현실적입니다.

### 3단계: 다음 분기 실행 마케팅 전략 도출

| 전략 | 실행 내용 (다음 분기) | S, T 요소 활용/대응 |
| :--- | :--- | :--- |
| 1. '고객 지지층' 마케팅 캠페인 | 충성도 높은 초기 사용자(S)를 '명예 앰버서더'로 선정하고, 이들이 직접 경쟁사 앱 대비 우리 앱의 '경험 가치'를 소셜 미디어에 공유하도록 파격적인 리워드를 제공합니다. | S(커뮤니티) 활용 T(가격)와 다른 '가치' 강조 |
| 2. '탈(脫) 가격' 소셜 콘텐츠 제작 | "저렴하지만 외로운 앱 vs 비싸도 함께 성장하는 앱"과 같이, 경쟁사의 가격 공세(T)를 직접 언급하며 '함께하는 즐거움'이라는 우리의 커뮤니티 강점(S)을 대비시키는 숏폼 콘텐츠를 제작합니다. | T(가격) 정면 대응 S(커뮤니티) 집중 어필 |
| 3. '커뮤니티 온보딩' 최적화 | 신규 사용자가 앱 설치 후 3분 이내에 최소 1개의 커뮤니티에 가입하거나, 다른 사용자로부터 환영 메시지를 받도록 자동화된 온보딩 프로세스를 구축합니다. 이는 경쟁사 앱 이탈 사용자에게 강력한 잔류 요인(S)으로 작용하게 합니다. | S(커뮤니티) 극대화 W(낮은 인지도)를 초기 경험으로 만회 |
