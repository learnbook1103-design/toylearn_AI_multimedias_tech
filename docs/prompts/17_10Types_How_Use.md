## 문제 상황
```
페르소나: 미디어 에이전시의 '멀티미디어 콘텐츠 매니저'

배경: 당신은 최근 대형 식품 기업의 신규 '친환경 밀키트' 브랜드 런칭 캠페인을 담당하게 되었습니다. 클라이언트(식품 기업)는 제품의 개발 과정, 재료의 신선함, 조리 편의성을 강조하는 **15분 분량의 고화질 홍보 영상(원본 소스)**을 대량으로 제작하여 전달했습니다.

핵심 문제: 클라이언트는 이 15분짜리 원본 영상 소스를 활용하여 유튜브(롱폼), 인스타그램(릴스/캐러셀), 틱톡, 네이버 블로그 등 모든 소셜 채널에 동일하게 배포하기를 원합니다. 하지만 당신은 플랫폼별 특성과 사용자 소비 행태가 완전히 다르다는 것을 알고 있습니다. 15분짜리 영상을 릴스나 틱톡에 그대로 올릴 수는 없으며, 블로그에는 영상과 다른 텍스트 중심의 콘텐츠가 필요합니다.

해결 과제: 클라이언트의 예산은 한정되어 있어 추가 촬영은 불가능합니다. 당신은 오직 제공된 15분짜리 원본 영상(과 그 스크립트)만을 활용하여, 각기 다른 4개 플랫폼(유튜브, 인스타그램 릴스, 틱톡, 네이버 블로그)의 성격에 최적화된 '멀티미디어 콘텐츠 배포 전략'을 수립하고, 각 플랫폼에 맞는 콘텐츠 변형(Repurposing) 기획안을 구체적으로 생성해야 합니다.
```

## 문제 해결 프롬프트
```

[문제 상황 및 제약 조건]

4.  예산 (제약): 콘텐츠 재가공 및 배포 예산이 한정되어 있어, 최소한의 자원으로 최대의 효과를 내는 전략이 필요합니다.
```
