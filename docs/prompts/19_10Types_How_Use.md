## 문제 상황
```
페르소나: 광고 대행사의 크리에이티브 디렉터 (Creative Director)
배경: 당신은 최근 클라이언트로 합류한 'AI 기반 업무 자동화 SaaS' 스타트업의 신규 캠페인 런칭을 총괄하게 되었습니다. 
클라이언트로부터 20페이지 분량의 상세한 **기술 명세서(Technical Specification)**와 기능 리스트를 전달받았습니다.
핵심 문제: 주 타겟 고객은 MZ세대 직장인입니다. 하지만 이 제품은 기능이 복잡하고 기술 용어가 많아, 기존의 텍스트 중심 설명 자료가 "너무 지루하고 복잡하다"라는 내부 테스트 피드백을 받았습니다.
 제품의 복잡한 기능을 직관적이고 '힙하게' 전달할 새로운 멀티미디어 접근 방식이 절실한 상황입니다.

해결 과제: 클라이언트가 제공한 20페이지 분량의 기술 명세서를 기반으로, 타겟 고객의 관심을 즉각적으로 사로잡을 수 있는 30초 분량의 '인스타그램 릴스' 또는 '틱톡' 숏폼 영상 광고 시나리오 3가지를 구체적으로 도출해야 합니다. 
(각 시나리오는 핵심 카피, 비주얼 컨셉, 활용할 BGM 아이디어를 포함해야 합니다.)
```

## 문제 해결 프롬프트 (역할 부여 + 제약 조건 결합)
```
[페르소나: 광고 대행사의 크리에이티브 디렉터]

다음은 기능 리스트에 기반하여, MZ세대 직장인의 공감을 얻을 수 있도록 복잡한 기능을 단순화한 30초 숏폼 광고 시나리오 3가지입니다.

#### 시나리오 1: [드래그 앤 드롭으로 퇴근] – 노코드 자동화 강조

| 항목 | 상세 내용 |
| :--- | :--- |
| 핵심 카피 | 코딩 없이 드래그만 하세요. 당신의 반복 업무는 이미 자동 퇴근 중. |
| 참조된 기술 명세 | 드래그 앤 드롭 에디터, 반복문 (Loop), 조건 분기 (IF/THEN) |
| 비주얼 컨셉 | 도입 3초: 피곤한 직장인이 같은 업무(데이터 복사/붙여넣기)를 반복하는 모습. 4초~15초: 화면이 'ReactFlow 기반'의 노코드 빌더 UI로 전환되며, 복잡했던 업무 플로우가 드래그 앤 드롭 노드 3~4개로 빠르게 구성되고 '반복문(Loop)' 노드가 강조됨. 후반 15초: 주인공이 5시 정각에 여유롭게 퇴근하며 스마트폰으로 '워크플로우 실행 통계'를 확인. |
| BGM 아이디어 | Chill-hop 장르의 배경 음악 (반복 업무 시) -> 경쾌하고 리드미컬한 트렌디 팝 (자동화 및 퇴근 시). |

---

#### 시나리오 2: [업무 카오스 탈출] – 통합 및 AI 분석 강조

| 항목 | 상세 내용 |
| :--- | :--- |
| 핵심 카피 | 슬랙에서 욕설 감지, 지메일로 경고. 당신의 AI 비서는 이미 일하고 있습니다. |
| 참조된 기술 명세 | Slack 연동, Gmail 연동, 감정 분석, 텍스트 분류 |
| 비주얼 컨셉 | 도입 5초: 'Slack' 채널에서 부정적인 고객 메시지 발생. 6초: AI가 메시지를 즉시 '감정 분석'하고 '부정'으로 분류. 10초~25초: 자동 워크플로우가 실행되어 해당 메시지를 'Gmail'로 담당 팀장에게 즉시 경고 메일 발송. 후반 5초: '노션' 페이지에 해당 이슈가 '긴급' 태그와 함께 자동 등록. |
| BGM 아이디어 | 긴장감 있는 신스웨이브(문제 발생 시) -> 깔끔하고 빠른 비트의 퓨처 하우스(자동화 처리 시). |

---

#### 시나리오 3: [5분짜리 보고서] – 데이터 처리 및 리포팅 강조

| 항목 | 상세 내용 |
| :--- | :--- |
| 핵심 카피 | 엑셀 데이터 파싱, 그래프 변환, 리포트 완성까지. 클릭 한 번이면 끝. |
| 참조된 기술 명세 | Google Sheets 연동, JSON 파싱, CSV 처리, 커스텀 템플릿 생성 |
| 비주얼 컨셉 | 도입 5초: MZ세대 직장인이 Google Sheets의 복잡한 데이터와 씨름하며 지친 모습. 6초~15초: 앱 내에서 'Google Sheets 연동 노드'를 연결하고 'JSON 파싱', 'CSV 처리' 노드를 순차적으로 연결하는 모습 시각화. 16초~25초: '커스텀 템플릿'에 데이터가 자동으로 삽입되어, 5분 만에 리포트가 완성되는 과정. 후반 5초: 리포트가 'Google Drive'에 자동 저장되며 알림이 뜸. |
| BGM 아이디어 | 경쾌하고 밝은 얼터너티브 팝. (업무의 지루함 없이 속도감 있는 처리 과정을 강조). |
```
