## 문제 상황
```
페르소나: 광고 대행사의 콘텐츠 제작 팀장
배경: 당신은 클라이언트 'A'(신규 뷰티 브랜드)의 신제품 런칭을 위해 3가지 버전의 유튜브 범퍼 광고(6초) 영상을 제작하여 집행했습니다. 3가지 영상은 각각 다른 소구점(A: 성분 강조, B: 사용 후기, C: 할인 혜택)을 강조했습니다.
핵심 문제: 캠페인 2주차 데이터 분석 결과, A와 B 영상은 평균 클릭률(CTR) 2.5%, **구매 전환율 1.8%**를 기록하며 순항 중입니다. 하지만 **가장 많은 예산을 투입한 C 영상(할인 혜택 강조)의 CTR은 0.8%**에 불과하며, **구매 전환율은 0.1%**로 처참한 수준입니다. 특히 C 영상은 3초 이내 이탈률이 80%에 육박합니다.
해결 과제: 클라이언트는 C 영상의 즉각적인 중단과 예산 환불까지 고려하고 있습니다. 당신은 C 영상의 성과가 저조한 구체적인 원인을 분석하고, 이 영상을 즉시 수정(re-edit)하거나 대체할 수 있는 긴급 대응 방안을 내일 오전 회의까지 보고해야 합니다.
```

## 프롬프트 유형과 선택 이유
| 유형 | 선택 이유 |
| :--- | :--- |
| 역할 부여 (Persona) | '광고 대행사 콘텐츠 제작 팀장' 페르소나를 부여하여, 단순한 분석이 아닌 **클라이언트의 우려를 해소**할 수 있는 **책임감 있고 실무적인 긴급 보고서** 형태의 답변을 유도합니다. |
| 제약 조건 | \*\*'6초 범퍼 광고'\*\*라는 길이 제약, \*\*'3초 이탈률 80%'\*\*라는 치명적인 데이터 제약, 그리고 \*\*'즉각적인 수정 또는 대체'\*\*라는 시간 제약을 명시하여, **현실적인 단기 액션 플랜**을 요구합니다. |
| (CoT 방식 차용) | **문제의 원인을 3단계(기술적/콘텐츠/전략적)로 구분**하여 진단하는 논리적 과정을 강제하여, **감에 의존하지 않는 데이터 기반의 보고서**를 작성하게 합니다. |

## 작성 위해 추가 필요 요건

| 구분 | 필요 요건 (핵심 질문) | 이유 (왜 이 정보가 필요한가) |
| :--- | :--- | :--- |
| **I. 기술적/집행적 요인** | **1. C 영상의 노출 빈도 및 타겟 설정** | C 영상이 A/B 영상과 **동일한 타겟**에게 **같은 빈도**로 노출되었는지 확인해야 합니다. 만약 C만 잘못된 타겟에게 집중 노출되었다면, **노출 설정 오류**가 원인일 수 있습니다. |
| **II. 콘텐츠 시각적 요인** | **2. C 영상의 도입부 3초 시각적 구성** | 이탈률 80%는 **도입부 3초 내에 시청자의 관심을 완전히 잃었다**는 의미입니다. 도입부에 **제품이 명확히 등장**하는지, **자막이 충분히 읽히는지**, 또는 **부정적인 시각적 요소**가 있는지 확인해야 합니다. |
| **III. 전략적 메시지 요인** | **3. C 영상의 '할인 혜택' 소구 방식** | A/B 영상은 '성분'/'후기'로 **제품 가치**를 전달했지만, C는 \*\*'할인 혜택'\*\*만 강조합니다. 혹시 혜택이 **너무 작게 표시**되었거나, **경쟁사 대비 매력이 떨어지는 혜택**이었는지 확인해야 합니다. |

## 문제 해결 프롬프트 (역할 부여 + 제약 조건 결합)
```
당신은 광고 대행사의 '콘텐츠 제작 팀장'이며, 클라이언트 'A'에게 제출할 긴급 보고서를 작성해야 합니다.

[문제 상황 데이터]
- A/B 영상 (성분/후기 강조): CTR 2.5%, 구매 전환율 1.8% (순항)
- C 영상 (할인 혜택 강조): CTR 0.8%, 구매 전환율 0.1%, 3초 이탈률 80% (처참)
- 제약: C 영상의 즉각적인 수정(Re-edit) 또는 대체 방안을 내일 오전까지 보고해야 합니다.

[지시 사항 - 긴급 진단 및 대응 3단계]

당신의 전문적인 판단을 바탕으로, C 영상의 성과 저조 원인을 진단하고 긴급 대응 전략을 수립하십시오.

1단계: 긴급 원인 진단 (3초 이탈률 분석)
- C 영상의 '3초 이탈률 80%'라는 데이터가 의미하는 가장 유력한 1차 원인을 기술적/시각적/전략적 세 가지 관점에서 각각 1가지씩 제시하고, 이 중 가장 치명적인 원인을 선택하십시오.

2단계: 즉시 수정(Re-edit) 가능성 진단 및 방향
- A/B 영상의 성공 요인(성분/후기)을 바탕으로, C 영상의 6초 길이 내에서 할인 혜택 강조를 유지하면서 A/B 영상의 강점을 활용할 수 있는 '수정(Re-edit) 방향' 2가지를 구체적으로 제시하십시오.

3단계: 긴급 대체 전략 (클라이언트 안심 보고서)
- 클라이언트에게 즉시 보고할 'C 영상을 대체할 수 있는 긴급 마케팅 액션 플랜' 2가지를 제시하십시오. 이 방안은 A/B 영상의 성공 예산을 활용하며, C 영상의 남은 예산 소진 및 성과 만회에 초점을 맞춰야 합니다.
```

## 프롬프트 실행 결과

### 1단계: 긴급 원인 진단 (3초 이탈률 분석)

| 관점 | 유력한 1차 원인 | 선택 이유 |
| :--- | :--- | :--- |
| 기술적 | C 영상만 잘못된 타겟(전환 의도 낮은 타겟)에 과도하게 노출. | A/B와 달리 C만 CTR과 전환율이 극단적으로 낮아, 타겟팅 오류 가능성이 높음. |
| 시각적 | 6초 도입부 3초 내에 '할인 혜택'이 너무 작거나 늦게 제시됨. | 3초 이탈률 80%는 광고를 클릭하기도 전에 시청자가 메시지를 파악하지 못했거나 지루했다는 뜻. 특히 혜택이 명확하지 않았을 가능성. |
| 전략적 | '할인 혜택' 메시지가 브랜드 '신뢰'나 '가치'를 훼손함. | 뷰티 신제품은 가치가 중요하나, C는 가격만 강조해 저가 브랜드로 오해되어 시청자의 거부감(스킵)을 유발했을 수 있음. |

가장 치명적인 원인 선택: 전략적 원인: '할인 혜택' 메시지가 브랜드 가치를 훼손함.

이유: 범퍼 광고(6초)는 짧은 시간에 브랜드 인식을 결정합니다. A/B는 제품의 가치를 성공적으로 전달했으나, C는 오직 가격만으로 접근하여 신규 뷰티 브랜드로서의 신뢰를 구축하지 못하고 이탈을 초래했을 가능성이 가장 높습니다. 3초 이내에 가치 전달에 실패했기 때문입니다.

---

### 2단계: 즉시 수정(Re-edit) 가능성 진단 및 방향

C 영상 (할인 혜택 강조)의 수정 방향:

| 수정 방향 | 세부 실행 방안 (6초 내) | 활용하는 A/B 강점 |
| :--- | :--- | :--- |
| 1. '성분 결합형' 메시지 전환 | 도입 1초: 핵심 성분('프리미엄 원료')을 시각적으로 빠르게 제시. 2초~5초: 성분과 제품이 빠르게 등장하며 '할인 메시지'를 크게 제시. 6초: 구매 유도. | A 영상의 '성분 강조'를 도입부에 결합하여 가치 훼손을 방지하고 시청자의 주의를 즉시 확보. |
| 2. '후기 기반' 긴급 대체 (대비 효과) | 도입 1~3초: A/B 영상의 성공적인 사용 전/후 또는 극적인 후기 텍스트를 먼저 보여줌. 4초~6초: "이 놀라운 결과가 지금 50% 할인"이라는 메시지를 삽입. | B 영상의 '사용 후기'를 활용하여 제품의 효능을 먼저 입증한 후, 할인 혜택을 '보너스'처럼 느껴지게 포지셔닝. |

---

### 3단계: 긴급 대체 전략 (클라이언트 안심 보고서)

| 긴급 액션 플랜 | 목표 및 기대 효과 |
| :--- | :--- |
| 1. C 영상 즉시 일시 중지 및 A/B 예산 증액 투입 | C 영상의 남은 예산을 즉시 회수하여 성과가 검증된 A/B 영상에 70% 재배분합니다. 나머지 30% 예산은 C의 수정(Re-edit) 버전(수정안 1번) 테스트에 투입하여 성과의 즉각적인 만회를 시도합니다. |
| 2. 리타겟팅 캠페인에 C 영상 메시지 재활용 | C 영상의 할인 혜택 메시지는 신규 유입 고객에게는 매력이 없었지만, A/B 영상을 클릭했던 잠재 고객에게는 효과적일 수 있습니다. C의 메시지를 A/B 클릭 유저 대상의 리타겟팅용으로만 전환하여, 예산 환불 없이 남은 예산을 전략적으로 소진합니다. |
